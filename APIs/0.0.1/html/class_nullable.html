Detailed Description

template<typename T>
class Nullable< T >

Templated container holding a single value of type T.

The value can be set at construction time only. The default constructor creates a container with no value set.

This is useful for example for functions that may or may not return a value. Alternative would be a std::pair<bool,T> or maybe an exception.
