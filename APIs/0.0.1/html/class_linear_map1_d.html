Detailed Description

Helper class that can convert between two different linear coordinate systems in 1D.

The reason for making this a class is to offer a slight optimization when one needs to apply the same linear map or its inverse to multiple values. The optimization is achieved by precomputing and storing some intermediates, so that linear transformation is simply a multiplication and addition.

Naming conventions: x1,x2 refer to the source coordinate system y1,y2 refer to the destination coordinate system
